# Tutor Base — База задач ЕГЭ/ОГЭ

## Структура репозитория

```
tutor-base/
├── .github/workflows/compile.yml   # Авто-компиляция PDF при push
├── templates/
│   └── sheet.tex                   # Шаблон листа задач для ученика
├── cs_ege/
│   ├── task_23_graphs/
│   │   ├── tasks.tex               # Задачи по теме
│   │   └── tasks.pdf               # Авто-генерируется
│   └── task_24_strings/
│       ├── tasks.tex
│       └── tasks.pdf
└── README.md
```

## Система ID задач

Формат: `{предмет}_{номер задания}_{сложность}_{номер}`

- Предметы: `cs_ege`, `cs_oge`, `math_ege`, `math_oge`
- Сложность: `easy`, `med`, `hard`
- Пример: `cs_ege_23_easy_001`

## Уровни сложности

| Уровень | Описание |
|---------|----------|
| `easy` | Базовые задачи, исполнитель с 2 командами без доп. условий |
| `med`  | Средние задачи, условие на траекторию |
| `hard` | Сложные задачи, несколько условий, большой диапазон |

## Как добавить задачи

1. Открыть нужный `tasks.tex` в папке темы
2. Добавить блок задачи по шаблону (см. комментарий внутри файла)
3. Сделать `git push` — PDF обновится автоматически за 1–3 мин
